# Deadman Switch Setup Instructions

## Initial Configuration
1. Go to the **Config** tab in your Start9 interface
2. Configure your email provider (Gmail or custom SMTP)
3. For Gmail: Enable 2FA and generate an App Password
4. Save the configuration and restart the service

## Usage
1. Open the web interface from your Services page
2. Create an account and log in
3. Configure recipient emails and deadman settings
4. Activate the deadman switch

## Important Notes
- Test your configuration before relying on it
- Keep your email credentials secure
- Regular check-ins are required to prevent activation
